Modul: "Pay-It-Easy" Zahlungsmodul

Version: 1.2.0

Beschreibung: Zahlungsmodul zur Durchführung einer Zahlung über das "Pay It Easy" System

Getestet mit Shop-System: Magento ver. 1.7.0.2, 1.8.0.0, 1.8.1.0

1	Installation
1.1 Installation des Plug-Ins
	a) Laden Sie die Installationsdatei "magento_Pay-It-Easy_1.2.0.zip" auf dem Shop-Server in ein temporäres Verzeichnis (/tmp) hoch.
	b) Entpacken Sie die Datei in das /tmp Verzeichnis (unzip magento_Pay-It-Easy_1.2.0.zip)
	c) Kopieren Sie den Inhalt des Verzeichnises "copy_this" in das Root-Verzeichnis des Shops.
	d) Stellen Sie sicher, dass die neu erstellten Dateien und Verzeichnisse für den Shop lesbar/ausführbar sind (Berechtigung 755).

2 Konfiguration
2.1 Konfiguration des Plug-Ins im Shopsystem
	a) Öffnen Sie im Browser die Administrationsansicht des Shops und melden Sie sich als Administrator an.
	b) Öffnen Sie die Ansicht "System>Konfiguration". Scrollen Sie im Fenster nach unten und klicken Sie auf "Zahlungsweisen".
	c) Klicken Sie auf die Bezeichnung "Pay It Easy Base".
	d) Tragen Sie die Ihnen zugewiesene Händlerkennung in das Feld "Händlerkennung" ein
	e) Tragen Sie den Ihnen zugewiesenen MAC-Schlüssel in das Feld "MAC-Schlüssel" ein.
	f) Speichern Sie Ihre Änderungen.
